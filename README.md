# Камышинский опытный завод — Официальный сайт

## Описание
Официальный веб-сайт Камышинского опытного завода — производителя промышленного оборудования.

## Структура проекта
```
/workspace
├── index.html          # Главная страница (основной HTML)
├── css/
│   └── styles.css      # Все стили приложения
├── js/
│   └── app.js          # Основной JavaScript код
├── images/             # Изображения (для будущего использования)
├── fonts/              # Шрифты (для будущего использования)
├── 1.0                 # Оригинальный файл (архив)
└── README.md           # Этот файл
```

## Установка и запуск

### Локальная разработка
1. Откройте `index.html` в любом современном браузере
2. Для разработки с live-reload используйте локальный сервер:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (требуется установка)
   npx serve .
   ```

### Требования
- Современный браузер (Chrome, Firefox, Safari, Edge)
- Поддержка ES6+

## Разработка

### Добавление новых стилей
Отредактируйте `css/styles.css`. Стили организованы по секциям:
- Переменные CSS
- Компоненты (кнопки, карточки, формы)
- Утилиты
- Адаптивность

### Добавление JavaScript
Отредактируйте `js/app.js`. Код использует модульный подход с IIFE.

### Добавление изображений
Поместите файлы в папку `images/` и обновите пути в HTML.

## Деплой

### Статический хостинг
Сайт является статическим и может быть размещён на:
- GitHub Pages
- Netlify
- Vercel
- Любом веб-сервере (Nginx, Apache)

### Рекомендации для production
1. Минифицируйте CSS и JS
2. Настройте CSP заголовки
3. Включите gzip/brotli сжатие
4. Настройте кэширование
5. Используйте HTTPS

## Безопасность
- Content Security Policy (CSP) рекомендуется настроить на уровне сервера
- Формы защищены от базовых XSS атак
- Внешние ресурсы загружаются через HTTPS

## Доступность (A11y)
- Skip-link для навигации с клавиатуры
- ARIA-атрибуты для модальных окон
- Семантическая HTML разметка
- Контрастные цвета соответствуют WCAG

## Производительность
- Lazy loading готовность
- Минимальное количество внешних зависимостей
- CSS вынесен в отдельный файл
- JavaScript модульный и оптимизированный
