# KOZ_Web — Сайт Камышинского опытного завода

Прототип официального веб-сайта Камышинского опытного завода.

## Описание

Данный проект представляет собой одностраничный веб-сайт (landing page) для Камышинского опытного завода.

## Технологии

- **HTML5** — структура страницы
- **Tailwind CSS** — стилизация через CDN
- **Google Fonts** — шрифт Inter

## Структура проекта

```
/workspace/
├── 1.0          # Основной HTML-файл с прототипом сайта
└── README.md    # Документация проекта
```

## Запуск

Откройте файл `1.0` в любом современном браузере:

```bash
# Linux/Mac
xdg-open 1.0      # Linux
open 1.0          # Mac

# Windows
start 1.0
```

Или просто перетащите файл в окно браузера.

## Особенности

- Адаптивный дизайн (responsive)
- Современный UI на базе Tailwind CSS
- Поддержка мобильных устройств

## Лицензия

Проект является прототипом и предназначен для демонстрационных целей.
